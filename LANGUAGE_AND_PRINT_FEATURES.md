## ğŸ”§ Technical Implementation

### **Translation Architecture**

```typescript
// How translations work:
1. User selects language from dropdown
2. Language code saved to localStorage
3. Translation JSON file loaded from /public/locales/{lang}.json
4. useTranslation() hook provides t() function
5. Components use t('key.path', 'fallback') to get translated text
6. All UI updates automatically
```

### **Translation Hook Usage**

```typescript
import { useTranslation } from '@/lib/translations';

function MyComponent() {
  const { t, language, setLanguage } = useTranslation();

    <div>

  );
}
```
